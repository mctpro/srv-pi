# SRV-PI

Projet personnel d'apprentissage de l'administration système.

## Matériel

- Raspberry Pi 5
- Raspberry Pi OS
- Différents ordinateurs connectés sur le réseau local

## Objectifs

- Prendre de l'expérience en administration système
- Héberger des services personnels

## Services installés

- Pi-hole
- Service de monitoring (node_exporter, prometheus, grafana)
- Docker
- Portainer
- Dozzle
- Portail web personnel
- Samba
- Fail2Ban
- UFW

## État du projet

Étude d'un projet d'agents IA basé sur des flux RSS crypto
