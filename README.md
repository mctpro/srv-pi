# SRV-PI

Projet personnel d'apprentissage de l'administration système.

## Matériel

- Raspberry Pi 5
- Raspberry Pi OS

## Objectifs

- Prendre de l'expérience en administration système
- Héberger des services personnels

## Services installés

- Pi-hole
- Service de monitoring (node_exporter, prometheus, grafana)
- Docker
- Portainer
- Dozzle
- Portail web personnel

## État du projet

🟢 Pi-hole opérationnel
🟢 Monitoring opérationnel
🟢 Docker et Portainer opérationnels
🟢 Portail web personnel fonctionnel
🟡 Étude d'un projet d'agents IA basé sur des flux RSS crypto + création de l'interface web pour un utilisateur externe
